Lenguaje programmación dinámico creado por Ericsson para desarrollar aplicaciones altamente concurrentes y con resistencia a fallos y optimizado para trabajar con redes de telecomunicaciones.

Utiliza una máquina virtual (BEAM)

La sintaxis de erlang puede ser muy complicada, por eso se creó el lenguaje [[Elixir]] para tratar de facilitar la creación de aplicaciones que corran en la maquina virtual de Erlang.
